## Day 30 特徵選擇
1.  特徵需要適當的增加與減少，以提升精確度並減少計算時間
  -   增加特徵 : 特徵組合 (Day 28) ，群聚編碼 (Day 29) 
  -   減少特徵 : 特徵選擇 (Day 30)

2.  特徵選擇有三大類方法 
-  過濾法 (Filter) : 選定統計數值與設定門檻，刪除低於門檻的特徵
-  包裝法 (Wrapper) : 根據目標函數，逐步加入特徵或刪除特徵
-  嵌入法 (Embedded) : 使用機器學習模型，根據擬合後的係數，刪除係數低於門檻的特徵
